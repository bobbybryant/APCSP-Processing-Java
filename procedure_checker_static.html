Does My Procedure Count?

your Create Task four-piece self-check

Paste your Processing code and find out which of the four required pieces are in your program, and what's still missing. It checks structure only — it will not write or fix any code for you. That part stays yours.

The four pieces you need

PIECE 1

A procedure you wrote

One function — not setup, draw, or mousePressed — that has a parameter you actually use, an if, and a loop, all in the same body.

PIECE 2

A call to it

Somewhere your program runs that procedure by name, away from where you defined it.

PIECE 3

A list that stores data

An array or an IntList / StringList / FloatList that actually holds values.

PIECE 4

Using that list

Read it or loop through it — get(i), [i], .size() — not just declare it.

the one most people miss ↓ A procedure only counts when one function you wrote has a parameter you use, an if, and a loop — all together. A parameter on a function that only draws, or a loop that lives in draw(), does not check the box.
